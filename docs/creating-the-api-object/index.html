Creating the API Object

API Object#

In this example we are creating the entire payload object by supplying the following components:

  • The auth object
  • The options object
  • The functions array - an array of objects defining which functions to run [ {...}, {...}, ... ]
    1. Starting a session - using function S3D.session.start
    2. Setting the model - using function S3D.model.set
    3. Running an analysis - using function S3D.model.solve
example-request-object.js
const customRequest = {
auth: {
username: 'YOUR_SKYCIV_USERNAME',
key: 'YOUR_SKYCIV_API_KEY',
},
options: {
validate_input: true,
},
functions: [
{
function: 'S3D.session.start',
arguments: {
keep_open: false,
},
},
{
function: 'S3D.model.set',
arguments: {
s3d_model: 'MODEL_OBJECT',
},
},
{
function: 'S3D.model.solve',
arguments: {
analysis_type: 'nonlinear',
repair_model: true,
},
},
],
};
